# LatestSpigot

A simple Rust program that downloads and builds the latest Spigot

### Requirements:
| Minecraft | Recommended |
|-----------|-------------|
| < 1.17    | [Java 8]    |
| 1.17      | [Java 16]   |
| 1.17 >    | [Java 17]   |
